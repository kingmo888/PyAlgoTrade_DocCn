feed – 数据源基类¶

Feeds 是将时间序列抽象化 当这些在事件调度循环时，会推送一个新的可用数据。 When these are included in the event dispatch loop, they emit an event as new data is available. Feeds are also responsible for updating the pyalgotrade.dataseries.DataSeries associated with each piece of data that the feed provides.

当前位置是关于数据源基类的，要了解bar的数据源请查阅： barfeed – Bar数据来源 板块.

class pyalgotrade.feed.BaseFeed(maxLen)¶

基类：pyalgotrade.observer.Subject

数据源基类

参数:maxLen (int.) – 每一个 pyalgotrade.dataseries.DataSeries 驻留的最大数量。当队列已满，一旦添加新的条目进来，将会在相反位置删除对应数量的条目。

注解

这是一个基类，不应该直接被调用。

__contains__(key)¶

如果给定的 pyalgotrade.dataseries.DataSeries 类型的key有效时，返回True。

__getitem__(key)¶

返回给定key的 pyalgotrade.dataseries.DataSeries

getNewValuesEvent()¶

当一个新的值可用时，新的事件将被触发. 订阅一个可调用对象需要两个参数：

  1. 一个 datetime.datetime 实例。
  2. 新的值。

CSV 支持¶

class pyalgotrade.feed.csvfeed.Feed(dateTimeColumn, dateTimeFormat, converter=None, delimiter=', ', timezone=None, maxLen=1024)¶

基类：pyalgotrade.feed.csvfeed.BaseFeed

一个从格式化后的CSV文件加载数据的数据源。

参数:
  • dateTimeColumn (string.) – 带有datetime列的列名.
  • dateTimeFormat (string.) – datetime的转换格式. datetime.datetime.strptime 将用它对该列进行格式转换.
  • converter (function.) – 一个有两个参数（列名和值）的转换函数。它将转换字符串到其他格式。默认转换为float，如果转换失败则返回string原值。
  • delimiter (string.) – 字符串分隔符.
  • timezone (A pytz timezone.) – The timezone to use to localize datetimes. Check pyalgotrade.marketsession.
  • maxLen (int.) – The maximum number of values that each pyalgotrade.dataseries.DataSeries will hold. Once a bounded length is full, when new items are added, a corresponding number of items are discarded from the opposite end.
addValuesFromCSV(path)¶

Loads values from a file.

参数:path (string.) – The path to the CSV file.

from pyalgotrade.feed import csvfeed

feed = csvfeed.Feed("Date", "%Y-%m-%d")
feed.addValuesFromCSV("quandl_gold_2.csv")
for dateTime, value in feed:
    print(dateTime, value)
