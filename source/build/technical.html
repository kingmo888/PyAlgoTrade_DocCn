class pyalgotrade.technical.EventWindow(windowSize, dtype=<class 'float'>, skipNone=True)¶

基类：object

事件窗口类负责在一个移动滑窗中进行计算。

参数:
  • windowSize (int.) – 滑窗的大小。必须大于0。
  • dtype (data-type.) – 数组所需的数据类型
  • skipNone (boolean.) – 是否跳过空值。默认为True 。一般在滑窗中不含空值的情况下使用True。

注解

这是一个基类，不应该直接被调用.

getValue()¶

通过滑窗中的值来计算一个值

getValues()¶

Returns a numpy.array with the values in the window.

getWindowSize()¶

返回滑窗的大小

class pyalgotrade.technical.EventBasedFilter(dataSeries, eventWindow, maxLen=1024)¶

基类：pyalgotrade.dataseries.SequenceDataSeries

EventBasedFilter 类负责接收一个 pyalgotrade.dataseries.DataSeries 类型的新值 并通过 EventWindow 来计算一个新值。

参数:
  • dataSeries (pyalgotrade.dataseries.DataSeries.) – 需要被计算指标的DataSeries实例。
  • eventWindow (EventWindow.) – 用于计算新值的EventWindow实例。

:param 存放的最大值。当队列已满，一旦添加新的条目进来，将会在相反位置删除对应数量的条目。 :type maxLen: int.

例子¶

下面的这个例子演示了如何结合 EventWindow 和 EventBasedFilter 来计算一个指标：

from pyalgotrade import dataseries
from pyalgotrade import technical


# An EventWindow is responsible for making calculations using a window of values.
class Accumulator(technical.EventWindow):
    def getValue(self):
        ret = None
        if self.windowFull():
            ret = self.getValues().sum()
        return ret

# Build a sequence based DataSeries.
seqDS = dataseries.SequenceDataSeries()
# Wrap it with a filter that will get fed as new values get added to the underlying DataSeries.
accum = technical.EventBasedFilter(seqDS, Accumulator(3))

# Put in some values.
for i in range(0, 50):
    seqDS.append(i)

# Get some values.
print(accum[0])  # Not enough values yet.
print(accum[1])  # Not enough values yet.
print(accum[2])  # Ok, now we should have at least 3 values.
print(accum[3])

# Get the last value, which should be equal to 49 + 48 + 47.
print(accum[-1])

输出内容如下:

None
None
3.0
6.0
144.0

pyalgotrade.technical.cross.cross_above(values1, values2, start=-2, end=None)¶

Checks for a cross above conditions over the specified period between two DataSeries objects.

It returns the number of times values1 crossed above values2 during the given period.

参数:
  • values1 (pyalgotrade.dataseries.DataSeries.) – The DataSeries that crosses.
  • values2 (pyalgotrade.dataseries.DataSeries.) – The DataSeries being crossed.
  • start (int.) – The start of the range.
  • end (int.) – The end of the range.

注解

The default start and end values check for cross above conditions over the last 2 values.
