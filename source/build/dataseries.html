dataseries – 数据序列基类¶

数据序列是管理time-series数据的抽象类。

class pyalgotrade.dataseries.DataSeries¶

基类：object

Base class for data series.

注解

这是一个基类，不应该直接被调用.

__getitem__(key)¶

Returns the value at a given position/slice. It raises IndexError if the position is invalid, or TypeError if the key type is invalid.

__len__()¶

Returns the number of elements in the data series.

getDateTimes()¶

Returns a list of datetime.datetime associated with each value.

class pyalgotrade.dataseries.SequenceDataSeries(maxLen=1024)¶

基类：pyalgotrade.dataseries.DataSeries

一个在内存序列中存放的DataSeries.

参数:maxLen (int.) – 允许驻留的最大数量。当队列已满，一旦添加新的条目进来，将会在相反位置删除对应数量的条目。
append(value)¶

添加一个带dateTime的值.

注解

如果dateTime非None，它必须大于原队列最后一个值。

getMaxLen()¶

返回允许驻留的最大数量

setMaxLen(maxLen)¶

调整允许驻留的最大数量

pyalgotrade.dataseries.aligned.datetime_aligned(ds1, ds2, maxLen=1024)¶

Returns two dataseries that exhibit only those values whose datetimes are in both dataseries.

参数:
  • ds1 (DataSeries.) – 一个DataSeries实例.
  • ds2 (DataSeries.) – 一个DataSeries实例.
  • maxLen (int.) – The maximum number of values to hold for the returned DataSeries. Once a bounded length is full, when new items are added, a corresponding number of items are discarded from the opposite end.

class pyalgotrade.dataseries.resampled.ResampledBarDataSeries(dataSeries, frequency, maxLen=1024)¶

基类：pyalgotrade.dataseries.bards.BarDataSeries, pyalgotrade.dataseries.resampled.DSResampler

创建一个更大频率的BarDataSeries，并将其作为一个新值推送到被重采样的位置。

参数:
  • dataSeries (pyalgotrade.dataseries.bards.BarDataSeries) – 需要被重采样的DataSeries
  • frequency – 用于分组的秒级频率。必须大于0。
  • maxLen (int.) – 驻留的最大数量。当队列已满，一旦添加新的条目进来，将会在相反位置删除对应数量的条目。

注解

  • 支持的重采样频率范围:
    • 日线以下频率
    • 日线（bar.Frequency.DAY）
    • 月线（bar.Frequency.MONTH）
